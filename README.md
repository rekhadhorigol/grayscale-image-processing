### Grayscale Image Processing
 
Applies fundamental **image processing operations** on grayscale images using NumPy array manipulation.
 
**Operations Covered:**
- Upload image and convert to grayscale (PIL)
- Retrieve image dimensions, pixel data type, min/max pixel values
- Crop image and paste into a black-border canvas
- **Vertical flip** (np.flipud)
- **Horizontal flip** (np.fliplr)
- **Transpose** - swap rows and columns
- **90° Rotation** (np.rot90)
- **Inversion** -> 255 - image
- Brightness/darkness adjustment (np.clip)
- **Andy Warhol effect** - horizontal stacking of image and its inverse
 
**Tools & Libraries:** numpy, matplotlib, PIL
